# Project Setup Guide
This document explains how to set up the project on your local environment.

## 1. Install Git
If Git is not installed, download it from [Git](https://git-scm.com/downloads)

## 2. Clone the Repository
Choose a directory on your machine and clone the repository by opening a terminal in your chosen folder and typing
    ```git clone (repository_url)```

## 2. Install Python
Make sure you have **Python 3.10+** installed.
If Python is not installed, download it from [Python.org](https://www.python.org/downloads/).

## 3. Create a Virtual Environment
Create an isolated Python environment for dependencies with
    ```python -m (venv) (venv_name)```

## 4. Activate the Virtual Environment (for Windows)
Activate the virtual environment by typing
    ```(venv_name)\Scripts\Activate```

## 5. Install Project Dependencies
Once the virtual environment is active, install all required packages by running
    ```pip install -r requirements.txt```
